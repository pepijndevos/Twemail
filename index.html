Twemail by pepijndevos

Twitter <-> email proxy in Python

Twemail is a Twisted server that acts as a proxy between Twitter and your email application, by providing POP and SMTP implementations that interacts with Twitter. Twemail allows you to read and write tweets using your mail client. It also supports replying and retweeting.

Dependencies

python twisted oauth2 pyOpenSSL twitter-text-python

Install

Point your email application to `@twitteremail-pepijndevos.dotcloud.com` using your Twitter password. Use port 18069 for POP3 and 18070 for SMTP.
